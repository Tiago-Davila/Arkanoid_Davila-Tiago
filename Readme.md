# 🧱 Arkanoid - Juego en Godot

## 📖 Descripción

Este proyecto es una recreación del clásico juego **Arkanoid**, desarrollado en **Godot Engine**.
El jugador controla una paleta que rebota una pelota con el objetivo de destruir todos los ladrillos en pantalla sin dejar que la pelota caiga.

El juego incluye mecánicas básicas como colisiones, rebotes, destrucción de objetos y condiciones de victoria/derrota.

---

## ⚙️ Instalación

1. Descargar o clonar el repositorio:

   ```bash
   git clone https://github.com/tu-usuario/arkanoid-godot.git
   ```

2. Abrir Godot Engine

3. Seleccionar **"Import"** y elegir la carpeta del proyecto

4. Ejecutar la escena principal (`main.tscn` o equivalente)

---

## 🚀 Características

* 🟦 Movimiento de la paleta controlado por el jugador
* ⚪ Pelota con rebotes dinámicos
* 🧱 Sistema de destrucción de ladrillos
* 🏆 Detección de victoria al eliminar todos los ladrillos
* 💀 Pantalla de derrota al perder la pelota
* ⚡ Aumento progresivo de velocidad de la pelota
* 🧠 Uso de lógica de colisiones y físicas del motor

---

## 🛠️ Tecnologías utilizadas

* 🎮 **Godot Engine**
* 🧾 **GDScript**
* 🧩 Sistema de nodos (`Node2D`, `RigidBody2D`, `Area2D`, `StaticBody2D`)
* 💥 Motor de físicas 2D de Godot

---

## 🎯 Objetivo del proyecto

Este proyecto fue realizado con fines educativos, para comprender:

* Uso del motor Godot
* Manejo de físicas en 2D
* Organización de escenas y nodos
* Lógica de videojuegos clásicos
